How to View PostScript and PDF files


PostScript:
You can view and print PostScript files using a program called Ghostview. Ghostview is available on most Unix systems. If you are a PC user and want to download Ghostview, click here. Note that probably the easiest way to view the files on a PC under Windows is to use the PDF format.

Compressed PostScript:
Is simply PostScript compressed with the gzip program. Use gunzip, which is available on most Unix systems to uncompress.

PDF:
This is the new Portable Display Format (Version 3.0) from Adobe. This is probably the easiest one to use in case you work on a PC with Windows. You can download version 3.0 or higher of Acrobat, a PDF viewer, for free from Adobe.
The latest version of Ghostview now also reads PDF files.
